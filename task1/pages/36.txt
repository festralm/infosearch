			<div><body dir="ltr" id="ibrwe2159f50" class="AppleTopic apd-topic dark-mode-enabled" data-istaskopen="true" data-designversion="2" lang="ru"><a name="ibrwe2159f50"></a><figure class="topicIcon"><img src="https://help.apple.com/assets/61DC93F5E0C8F948D777EC35/61DC93F5E0C8F948D777EC3C/ru_RU/6846073fc6f766d1ad3f66ae757c5e24.png" alt="" height="30" width="30" originalImageName="SharedGlobalArt/AppIconTopic_Safari_11.png"></figure><h1>Изменение настроек веб-сайтов в Safari на компьютере Mac</h1><p>В приложении Safari <img src="https://help.apple.com/assets/61DC93F5E0C8F948D777EC35/61DC93F5E0C8F948D777EC3C/ru_RU/6846073fc6f766d1ad3f66ae757c5e24.png" alt="" height="30" width="30" originalImageName="SharedGlobalArt/AppIconTopic_Safari_11.png"> на компьютере Mac в разделе «Веб-сайты» можно настроить параметры просмотра отдельных веб-сайтов. Чтобы изменить эти настройки, выберите меню «Safari» &gt; «Настройки» и нажмите «Веб-сайты».</p><p>Параметры, которые можно настроить (например, режим «Для чтения» и правила блокирования контента) перечислены слева.</p><p>Чтобы применить настройки параметра для веб-сайта справа, необходимо выбрать этот параметр, а затем выбрать нужный вариант из всплывающего меню рядом с веб-сайтом.</p><p><a href="x-help-action://openApp?bundleId=com.apple.Safari" class="URL">Открыть Safari</a></p><div class="table-wrapper"><table data-type="Multicolumn" aria-label="Описания параметров в панели «Веб-сайты» в настройках Safari"><thead><tr><th colspan="4" scope="col"><p class="TableHead">Параметр</p></th><th colspan="8" scope="col"><p class="TableHead">Описание</p></th></tr></thead><tbody><tr><td colspan="4"><p>Для чтения</p></td><td colspan="8"><ul><li><p><em>Вкл.</em> Автоматически включать режим «Для чтения» в Safari, чтобы отображать статьи на веб-сайте без рекламы и других отвлекающих элементов.</p></li><li><p><em>Выкл.</em> По умолчанию не использовать «Для чтения» на этом сайте. (Можно включить режим «Для чтения» для отдельных статей на этом сайте вручную.)</p></li></ul></td></tr><tr><td colspan="4"><p>Правила блокирования контента</p></td><td colspan="8"><ul><li><p><em>Вкл.</em> Не показывать рекламу и другой нежелательный контент на сайте.</p></li><li><p><em>Выкл.</em> Не блокировать рекламу и другой нежелательный контент на сайте.</p></li></ul></td></tr><tr><td colspan="4"><p>Автовоспроизведение</p></td><td colspan="8"><ul><li><p><em>Всегда.</em> Все видео на сайте могут воспроизводиться автоматически.</p></li><li><p><em>Только для файлов без звука.</em> Только видео без звука будут воспроизводиться автоматически.</p></li><li><p><em>Никогда.</em> Видео не будут воспроизводиться автоматически.</p></li></ul></td></tr><tr><td colspan="4"><p>Масштаб страницы</p></td><td colspan="8"><p>Выбрать увеличение масштаба (в процентах), чтобы текст и изображения на сайте были лучше видны.</p></td></tr><tr><td colspan="4"><p>Камера</p></td><td colspan="8"><ul><li><p><em>Спрашивать.</em> Сайт должен запрашивать разрешение, чтобы использовать камеру на Mac.</p></li><li><p><em>Отказать.</em> Сайт не может использовать камеру.</p></li><li><p><em>Разрешить</em> Сайт всегда может использовать камеру.</p></li></ul></td></tr><tr><td colspan="4"><p>Микрофон</p></td><td colspan="8"><ul><li><p><em>Спрашивать.</em> Сайт должен запрашивать разрешение, чтобы использовать микрофон на Mac.</p></li><li><p><em>Отказать.</em> Сайт не может использовать микрофон.</p></li><li><p><em>Разрешить</em> Сайт всегда может использовать микрофон.</p></li></ul></td></tr><tr><td colspan="4"><p>Общий экран</p></td><td colspan="8"><ul><li><p><em>Спрашивать.</em> Сайт должен запрашивать разрешение, чтобы использовать камеру и микрофон на Mac.</p></li><li><p><em>Отказать.</em> Сайт не может использовать камеру и микрофон.</p></li></ul></td></tr><tr><td colspan="4"><p>Размещение</p></td><td colspan="8"><ul><li><p><em>Спрашивать.</em> Сайт должен запрашивать разрешение, чтобы использовать Вашу геопозицию.</p></li><li><p><em>Отказать.</em> Сайт не может использовать Вашу геопозицию.</p></li><li><p><em>Разрешить</em> Сайт всегда может использовать Вашу геопозицию.</p></li></ul></td></tr><tr><td colspan="4"><p>Загрузки</p></td><td colspan="8"><ul><li><p><em>Спрашивать.</em> Сайт должен запрашивать разрешение, чтобы начать загрузку.</p></li><li><p><em>Отказать.</em> Сайт не может выполнять загрузку на Ваше устройство.</p></li><li><p><em>Разрешить</em> Сайт всегда может выполнять загрузку на Ваше устройство.</p></li></ul></td></tr><tr><td colspan="4"><p>Уведомления</p></td><td colspan="8"><ul><li><p><em>Разрешить</em> Сайт всегда может отправлять Вам уведомления.</p></li><li><p><em>Отказать.</em> Сайт не может отправлять Вам уведомления.</p></li><li><p><em>Разрешить веб-сайтам запрашивать разрешение на отправку уведомлений.</em> При посещении веб-сайта, который хочет отправлять уведомления, отображается диалоговое окно с вопросом: хотите ли Вы их получать. Чтобы это диалоговое окно больше не открывалось, снимите этот флажок.</p></li></ul></td></tr><tr><td colspan="4"><p>Всплывающие окна</p></td><td colspan="8"><p>Можно блокировать всплывающие окна, которые появляются при открытии или закрытии веб-страниц.</p><ul><li><p><em>Блокировать и оповещать.</em> Когда Вы открываете сайт, Safari отображает значок в <a href="https://support.apple.com/ru-ru/guide/safari/aside/glos5a7c4a65/14.0/mac/11.0" class="xRef Aside">поле смарт-поиска</a>. Вы можете нажать этот значок, чтобы отобразить всплывающие окна.</p></li><li><p><em>Заблокировать.</em> Сайт не может отображать всплывающие окна.</p></li><li><p><em>Разрешить</em> Сайт может отображать всплывающие окна.</p></li></ul><div class="Alert"><p class="Note"><em>Примечание. </em>Некоторые веб-сайты используют всплывающие окна для отображения важной информации.</p></div></td></tr><tr><td colspan="4"><p>WebGL</p></td><td colspan="8"><ul><li><p><em>Спрашивать.</em> Вы увидите диалоговое окно с вопросом: разрешаете ли Вы сайту использовать WebGL.</p></li><li><p><em>Заблокировать.</em> Сайт не может показывать содержимое WebGL.</p></li><li><p><em>Разрешить</em> Сайт может использовать WebGL, кроме случаев, когда WebGL представляет угрозу безопасности компьютера.</p></li><li><p><em>Разрешать всегда.</em> Сайт может использовать WebGL, даже если WebGL представляет критическую угрозу безопасности компьютера. Вариант «Разрешать всегда» создает определенные риски для безопасности, поэтому выбирать его следует только для тех веб-сайтов, которым Вы доверяете.</p></li></ul><p>Параметр «WebGL» отображается только для компьютеров, на которых WebGL может представлять угрозу безопасности. Если этот параметр не отображается, сайт может отображать содержимое WebGL.</p></td></tr><tr><td colspan="4"><p>Веб-сайты, открытые в данный момент </p></td><td colspan="8"><p>Веб-сайты, открытые в окне или вкладке браузера Safari. Для некоторых из этих веб-сайтов уже может быть выбран вариант во всплывающем меню.</p></td></tr><tr><td colspan="4"><p>Настроенные веб-сайты</p></td><td colspan="8"><p>Список настроенных Вами веб-сайтов. Если список «Настроенные веб-сайты» пуст, Вы еще не настроили ни одного веб-сайта или очистили список.</p></td></tr><tr><td colspan="4"><p>Удалить</p></td><td colspan="8"><p>Удаление выбранного веб-сайта из списка настроенных веб-сайтов.</p></td></tr><tr><td colspan="4"><p>При посещении других веб-сайтов</p></td><td colspan="8"><p>В этом всплывающем меню можно выбрать параметр, чтобы применить его к любому веб-сайту, для которого еще не задана настройка.</p><p>Чтобы применить настройку ко всем веб-сайтам, убедитесь, что список «Настроенные веб-сайты» пуст (чтобы быстро очистить этот список, выберите веб-сайты и нажмите «Удалить»). Затем нажмите это всплывающее меню и выберите нужный вариант.</p></td></tr></tbody></table></div><div class="LinkUniversal"><strong>См. также</strong><a href="https://support.apple.com/ru-ru/guide/safari/ibrw7f78f7fe/14.0/mac/11.0" class="xRef AppleTopic">Настройка параметров для каждого веб-сайта отдельно в Safari на компьютере Mac</a><a href="https://support.apple.com/ru-ru/guide/safari/sfri35905/14.0/mac/11.0" class="xRef AppleTopic">Ведение списка для чтения в Safari на компьютере Mac</a><a href="https://support.apple.com/ru-ru/guide/safari/ibrw1068/14.0/mac/11.0" class="xRef AppleTopic">Увеличение веб-страниц в Safari на компьютере Mac</a><a href="https://support.apple.com/ru-ru/guide/safari/sfri40734/14.0/mac/11.0" class="xRef AppleTopic">Настройка уведомлений от веб-сайтов в Safari на компьютере Mac</a></div></body></div>
