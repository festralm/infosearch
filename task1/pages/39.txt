<article class="ocpArticleContent"><section class="ocpIntroduction">
        <p>Расширения и дополнения — это простые инструменты, позволяющие настраивать интерфейс браузера и предоставляющие вам дополнительные возможности управления. В этой статье приведены инструкции по добавлению расширений в новый браузер Microsoft Edge <span class="icon-fluent icon-edgelogo"></span>, а также их отключению и удалению из браузера.</p>
      </section><section class="ocpSection" aria-labelledby="ID0EDN">
        <h2 id="ID0EDN">Поиск и добавление расширения в Microsoft Edge</h2>
        
          <ol type="1" itemscope="" itemtype="http://schema.org/ItemList">
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>В Microsoft Edge выберите <b class="ocpUI">Расширения</b> справа от адресной строки браузера.  <br><br> <img src="https://support.content.office.net/ru-ru/media/569cb076-8f8f-4145-afc9-e38ee65714ed.png" alt="Управление расширениями на панели браузера Microsoft Edge." loading="lazy"></p>
              <div class="ocpAlert">
        <p class="ocpAlertSection"><b class="ocpNote">Примечание: </b>Если пункт <b class="ocpUI">Расширения</b> не отображаются справа от адресной строки браузера, выберите <b class="ocpUI">Настройки и прочее </b><span class="icon-fluent icon-more"></span> &gt; <b class="ocpUI">Расширения</b>.</p>
      </div>
            <meta itemprop="position" content="1">
    </li>
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>Выберите <b class="ocpUI">Открыть надстройки Microsoft Edge</b>.</p>
            <meta itemprop="position" content="2">
    </li>
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>Найдите расширение, которое нужно добавить, и нажмите кнопку <b class="ocpUI">Получить</b>.</p>
            <meta itemprop="position" content="3">
    </li>
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>При отображении запроса со списком разрешений, необходимых расширению, внимательно изучите их и, чтобы продолжить, нажмите кнопку <b class="ocpUI">Добавить расширение</b>.</p>
            <meta itemprop="position" content="4">
    </li>
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>Появится последний запрос, подтверждающий добавление расширения.</p>
            <meta itemprop="position" content="5">
    </li>
  </ol>
        
      </section><section class="ocpSection" aria-labelledby="ID0EDL">
        <h2 id="ID0EDL">Добавление расширения в Microsoft Edge из интернет-магазина Chrome</h2>
        
          <p>Расширения, предназначенные для Google Chrome, также можно использовать в Microsoft Edge.</p>
          <ol type="1" itemscope="" itemtype="http://schema.org/ItemList">
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>В Microsoft Edge перейдите в <a href="https://chrome.google.com/webstore/category/extensions" target="_blank" class="ocpExternalLink">интернет-магазин Chrome</a>.</p>
            <meta itemprop="position" content="1">
    </li>
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>Выберите <b class="ocpUI">Разрешить расширения из других магазинов</b> в заголовке в верхней части страницы, а затем выберите <b class="ocpUI">Разрешить</b>, чтобы подтвердить.</p>
            <meta itemprop="position" content="2">
    </li>
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>Выберите расширение, которое требуется добавить, и нажмите <b class="ocpUI">Добавить в Chrome</b>.</p>
            <meta itemprop="position" content="3">
    </li>
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>При отображении запроса со списком разрешений, необходимых расширению, внимательно изучите их и, чтобы продолжить, нажмите кнопку <b class="ocpUI">Добавить расширение</b>.</p>
            <meta itemprop="position" content="4">
    </li>
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>Появится последний запрос, подтверждающий добавление расширения.</p>
            <meta itemprop="position" content="5">
    </li>
  </ol>
        
      </section><section class="ocpSection" aria-labelledby="ID0EDJ">
        <h2 id="ID0EDJ">Отключение расширения в Microsoft Edge</h2>
        
          <ol type="1" itemscope="" itemtype="http://schema.org/ItemList">
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>В Microsoft Edge выберите <b class="ocpUI">Расширения</b> справа от адресной строки браузера, а затем выберите <b class="ocpUI">Управление расширениями</b>.</p>
            <meta itemprop="position" content="1">
    </li>
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>Выберите переключатель рядом с расширением, которое нужно отключить.</p>
            <meta itemprop="position" content="2">
    </li>
  </ol>
        
      </section><section class="ocpSection" aria-labelledby="ID0EDH">
        <h2 id="ID0EDH">Удаление расширения из Microsoft Edge</h2>
        
          <p>Чтобы удалить любое расширение, выберите один из следующих методов. </p>
          <ul>
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>В Microsoft Edge выберите и удерживайте (или щелкните правой кнопкой мыши) значок расширения, которое нужно удалить (справа от адресной строки браузера). Выберите <b class="ocpUI">Удалить из Microsoft Edge </b>&gt;<b class="ocpUI"> Удалить</b>.</p>
            <meta itemprop="position" content="1">
    </li>
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>Справа от адресной строки браузера выберите <b class="ocpUI">Расширения</b>, а затем выберите <b class="ocpUI">Дополнительные действия </b><span class="icon-fluent icon-more"></span> рядом с расширением, которое нужно удалить. Затем выберите <b class="ocpUI">Удалить из Microsoft Edge </b>&gt;<b class="ocpUI"> Удалить</b>.</p>
            <meta itemprop="position" content="2">
    </li>
  </ul>
        
      </section><section class="ocpSection" aria-labelledby="ID0EDF">
        <h2 id="ID0EDF">Сообщение о нарушении</h2>
        
          <p>Если вы обнаружите на веб-сайте Надстройки Microsoft Edge что-то, что нарушает <a href="https://docs.microsoft.com/microsoft-edge/extensions-chromium/store-policies/ada-addendum" target="_blank" class="ocpExternalLink">политики в отношении надстроек Microsoft Edge</a>, содержит вредоносные программы, нарушает авторские права или является оскорбительным или неприятным, сообщите нам об этом.</p>
          <ol type="1" itemscope="" itemtype="http://schema.org/ItemList">
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>Войдите на <a href="https://microsoftedge.microsoft.com/addons/Microsoft-Edge-Extensions-Home" target="_blank" class="ocpExternalLink">сайт надстроек Microsoft Edge</a>.</p>
            <meta itemprop="position" content="1">
    </li>
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>Выберите расширение, о котором вы хотите сообщить.</p>
            <meta itemprop="position" content="2">
    </li>
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>Справа от открывающегося окна выберите <b class="ocpUI">Сообщить о нарушении</b>.</p>
            <meta itemprop="position" content="3">
    </li>
    <li itemprop="itemListElement" itemscope="" itemtype="http://schema.org/ListItem">
              <p>Заполните форму и нажмите кнопку <b class="ocpUI">Отправить</b>.</p>
            <meta itemprop="position" content="4">
    </li>
  </ol>
        
      </section><section class="ocpSection" aria-labelledby="ID0EDD">
        <h2 id="ID0EDD">Оставьте оценку или отзыв</h2>
        
          <p>Чтобы оставить оценку или отзыв, откройте расширение на <a href="https://microsoftedge.microsoft.com/addons/Microsoft-Edge-Extensions-Home" target="_blank" class="ocpExternalLink">сайте надстроек Microsoft Edge</a>и нажмите кнопку <b class="ocpUI">Написать отзыв</b>.</p>
          <p>
            <a href="https://support.microsoft.com/microsoft-edge-legacy" target="_blank" class="ocpExternalLink">Получение справки по предыдущей версии Microsoft Edge</a>
          </p>
